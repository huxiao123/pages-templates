# React ESR 模板

ESR 是一种在边缘计算节点上执行的渲染技术，这些节点通常在地理位置上更接近用户，从而能够更快地响应用户请求。通过结合 SSR 和 CSR 的优势，ESR 可以在边缘节点上生成 HTML 页面并将其交付给客户端。

此模板是一个使用边缘函数渲染页面的 React 应用。
