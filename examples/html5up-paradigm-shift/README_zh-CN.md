你好，欢迎来到 Paradigm Shift，这是我一年多来首次为 HTML5 UP 添加的作品 (!!!)。这次我采取了一种稍微不同的方向，部分原因是想尝试 CSS grid（这次设计中大量依赖了它），但主要是想回归制作更奇怪和更具实验性的设计/布局。无论如何，希望你喜欢它，感谢你在没有更新的情况下仍然给予的持续支持 :)
